Для того чтобы создать пейджинг в галлерее нужен снипет getPage.

В версии getPage 1.2.3 некорректно работает с снипетом Gallery -- не происходит перехода по страницам.

Для того чтобы это исправить нужно внести правку в снипет getPage:

$properties['offset'] = (!empty($properties['limit']) &&
 !empty($properties['page'])) ? ($properties['limit'] *
 ($properties['page'] - 1)) : 0;
$properties['start'] = $properties['offset']; // добавить эту строку

После этого можно использовать такой сниппет:

[[!getPage?
&elementClass=`modSnippet`
&element=`Gallery`
&thumbWidth=`150`
&thumbHeight=`150`
&limit=`2`
&album=`1`
]]
[[!+page.nav]]
