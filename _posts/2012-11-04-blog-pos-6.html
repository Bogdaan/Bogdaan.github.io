Нужно было сделать фильтр для шин и дисков по автомобилю. Собственно базу соответствия "авто - размеры шин и дисков" в открытом доступе не нашел. Поэтому спарсил соответствия с уже существующих порталов:

  • a_brends - таблица брендов авто
  • a_models - таблица моделей авто
  • а_years  - таблица содержит год выпуска
  • a_mods - таблица модификаций 
  • a_tyre_sizes_dup -  таблица рекомендаций параметров шин для конкретной модели авто ( по бренду + модели + году выпуска + модификации )
  • a_whell_sizes_dup - таблица рекомендаций параметров дисков для конкретной модели авто ( по бренду + модели + году выпуска + модификации )

База неполная, нет рекомендаций типоразмеров для ВАЗ, ГАЗ, ТаГАЗ и некоторых других марок, были ошибки в парсере.
У кого есть более полный список - поделитесь.
